# 使用手册（教师 / 学生）

## 学生端
1. 登录后在「我的实验」选择实验，先完成课前测试。
2. 四阶段任务：概念理解(25) → AI 协同编程(30) → 程序分析(25) → 综合应用(20)。
3. AI 协同阶段可向课程智能体「小林同学」求助，系统采用「不直接给答案、以追问启发」的苏格拉底式辅导；可在线运行代码。
4. 提交后异步队列在数秒内返回分项反馈。
5. 「待答辩」卡片出现时，进入完成课前闭卷答辩，用于核验对自己作品的真实理解。

## 教师端
- **班级学情分析**：四阶段得分、薄弱点、AI 依赖画像。
- **AI 协同画像中心（AICI）**：点「重算 AICI（全部实验）」对历史提交批量回算；可下钻查看四分量明细与四级标签。
- **早期预警**：8 类规则（超时、高分零对话、极速完成、直接索取答案、课前薄弱、低质交互、静默超时等），可逐条处理/忽略。
- **答辩管理**：单条/批量生成答辩题，复核 AI 初评并改分，查看综合分与作品-答辩落差预警。
- **历史成绩归档**：按班级/学生查看迁移的历史成绩。
- **算力配置**：切换 local/cloud/hybrid，测试端点连通性。
